# TETU Core

A 4X game in space, inspired by Stellaris PDX

## Installation

Install `git`, `sfml`, `crystal`, `make`, `imgui`.

    make deps
    export LD_LIBRARY_PATH="$(pwd)/cimgui"
    make release
    export LD_LIBRARY_PATH="$(pwd)/lib/imgui-sfml"
    ./core

## Usage

    make run
